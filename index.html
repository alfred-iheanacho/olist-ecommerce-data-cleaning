Olist E-Commerce Data Cleaning Project

End-to-end data cleaning and preprocessing project using Python, Pandas and NumPy on the Olist Brazilian E-Commerce Dataset.

View GitHub Repository

Business Problem

Business datasets frequently contain missing values, duplicates, inconsistent formats and data quality issues. These problems can reduce the reliability of analytics, dashboards and machine learning models.

The objective of this project was to clean and standardize the Olist Brazilian E-Commerce Dataset to produce analysis-ready datasets suitable for reporting and business intelligence.

Dataset Overview

The Olist dataset contains transactional information from a Brazilian e-commerce marketplace.

  • Customers
  • Orders
  • Order Items
  • Payments
  • Products
  • Sellers
  • Reviews
  • Category Translations
